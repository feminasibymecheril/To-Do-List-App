# ✅ Taskora - Smart Task Manager

## 📌 Project Description
Taskora is a **modern and interactive task management web application** built using **HTML, CSS, and Vanilla JavaScript**.

It helps users efficiently manage daily tasks with features like:
- Task creation and editing
- Progress tracking
- Filtering and search
- Drag-and-drop reordering
- Dark mode

All data is stored locally using **Local Storage**, so tasks persist even after refreshing the page.

---

## 🚀 Features

### 📝 Task Management
- Add new tasks with:
  - Title
  - Description
  - Priority (Low, Medium, High)
  - Due Date
  - Category
- Edit existing tasks
- Delete tasks

### ✅ Task Status
- Mark tasks as completed/pending
- Visual indication for completed tasks

### 🔍 Search & Filter
- Search tasks in real-time
- Filter tasks:
  - All
  - Completed
  - Pending

### 📊 Progress Tracking
- Dynamic progress bar
- Shows percentage of completed tasks
- Task counter display

### 🎯 Drag & Drop
- Reorder tasks using drag-and-drop

### 🌙 Dark Mode
- Toggle between light and dark themes

### 💾 Local Storage
- Tasks are saved in browser storage
- Data persists after refresh

## 🧠 Core Functionalities

### Task Operations
- `addTask()` → Creates new task  
- `updateTask()` → Edits existing task  
- `deleteTask()` → Removes task  
- `toggleComplete()` → Marks task complete/incomplete  

### Rendering
- `renderTasks()` → Displays tasks dynamically  
- Applies filters and search together  

### Storage
- `saveToLocalStorage()` → Saves tasks  
- Loads tasks automatically on page load  

### Drag & Drop
- Allows reordering tasks visually  
- Updates task list accordingly
